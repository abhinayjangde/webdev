There are two type of tags:

Semantic Tags

Semantic tags are those tags which clearly defines its content. For example: <form>, <table>, <article>, <aside>, <details>, <figcaption>, <figure>, <footer>, <header>, <main>, <mark>, <nav>, <section>, <summary>, <time>

Non-Semantic Tags

Non-Semantic tags are those tags which doesn't clearly defines its content. For example: <div>, <span>
